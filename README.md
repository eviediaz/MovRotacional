# MovRotacional

Es un proyecto realizado en Arduino para en realizar una plataforma de soporte para una cámara. 
Dicha base contará con movimiento esférico, es decir, dos tipos de giro: vertical y horizontal. 
Para controlar este movimiento, habrá un panel adjunto con dos opciones de movimiento, rutas predefinidas y control por joystick. 
Además, se le agregó una funcionalidad Bluetooth que permite el movimiento vertical y horizontal mediante un aplicativo.
